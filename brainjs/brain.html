Prediction

So, what exactly are we looking at? This background color of this square is a randomly generated color, 0-1 multiplied by 255 for each r, g, b value.

The text in the center is either white, or black depending on a model prediction to maximize text contrast for readability.

I trained the model with the first data points of r(0) g(0) b(0) should output black, and r(1) g(1) b(1) should output white. I then did some manual training - clicking the button for which color the text should be for the given background adds the rgb color of the current state as input, and my determination of text color for output into the test set of the model. If you print and observe the console output, you can see an example of this.

If you wish to further train the model, you can use the white/black buttons to select what you believe it should have displayed for the given color, and this instance will be added to the model's test set, or just refresh the page or click print to see more examples of my model predicting the correct text output.
